## Getting started

Prerequisites:

- [cargo](https://www.rust-lang.org/tools/install)
- [docker (compose)](https://docs.docker.com/engine/install/)


#### Initialize

Run `cargo xtask --help` for a list of the other xtask commands

``` bash
cargo xtask init
```



#### running the API

``` bash
cargo run
```
