A cleanly designed Rust REST Api


## Getting started

**Prerequisites:**

- [cargo](https://www.rust-lang.org/tools/install)
- [cargo-make](https://github.com/sagiegurari/cargo-make#installation)
- [docker (compose)](https://docs.docker.com/engine/install/)

#### 1 - Initialize

``` bash
cargo make init
```

#### 2 - Run API

``` bash
cargo make run
```
